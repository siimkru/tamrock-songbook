# ChordPro syntax documentation

This directory is a snapshot of the official ChordPro file-format
documentation. Start with:

- `ChordPro-Cheat_Sheet.md` for a compact reference.
- `ChordPro-Introduction.md` for the format basics.
- `ChordPro-Directives.md` and the `Directives-*.md` files for directives.
- `ChordPro-Chords.md` for chord syntax and parsing.
- `ChordPro-Markup.md` for inline text formatting.

Source: <https://github.com/ChordPro/chordpro/tree/master/docs/content>
